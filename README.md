## About The Project

This project is a modern starter application powered by:

- **Laravel 13** — Elegant and powerful backend framework.
- **TailwindCSS** — Utility-first CSS framework for rapid UI development.
- **FluxUI** — Beautiful UI components for Laravel & Livewire applications.

The goal of this project is to provide a scalable, clean, and developer-friendly foundation for building modern web applications.

# Installation

---

## Install Dependency

Install dependency PHP:

```bash
composer install
```

Install dependency JavaScript:

```bash
npm install
```
```bash
npm run build
```

---

## Environment Setup

Copy file environment:

```bash
cp .env.example .env
```

Generate application key:

```bash
php artisan key:generate
```

---

## Database Migration

Jalankan migration:

```bash
php artisan migrate
```

---

## Install Package

Maat Website Excel

```bash
composer require maatwebsite/excel
```
